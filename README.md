# Customer Churn Prediction

## Problem
Customer churn leads to significant revenue loss. The goal of this project is to identify customers who are likely to leave a company.

## Data
The dataset includes customer information such as:
- Age
- Balance
- Geography
- Activity status

Target variable:
- Churn (0 = stayed, 1 = left)

## Process
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training using Logistic Regression, Random Forest, and CatBoost
- Model evaluation using F1 score and ROC-AUC
